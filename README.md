# XMind Viewer for VS Code

A simple extension to view `.xmind` files directly in Visual Studio Code.

## Features

- **Read-only Preview**: Open valid `.xmind` files to see your mind map structure.
- **Interactive**: Support zoom, drag, and node collapse/expand.

## How to use

1. Right click on any `.xmind` file in the explorer.
2. Select files to open, or double click if it's set as default.

## Requirements

None.

## Known Issues

- Only supports basic node text rendering. Complex styles or attachments might not be fully supported.

**Enjoy!**
