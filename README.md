# XMind Viewer & Editor for VS Code

这是一款深度集成于 Visual Studio Code 的 **XMind** 文件编辑神器。它采用“原生化”设计理念，为您提供无论是交互手感还是视觉审美都媲美官方软件的卓越体验。

无需离开 VS Code，即可流畅地查看、编辑、导出您的思维导图，让知识管理与代码开发无缝融合。

---

## 🔥 核心特性 (Key Features)

### 1. 沉浸式编辑体验
- **原生级渲染**：完美还原 XMind 原生样式，支持线条、边框、字体颜色的精准呈现。
- **深色模式适配**：自动跟随 VS Code 主题切换明/暗色模式，夜晚编码也能护眼。
- **极简交互设计**：采用 Flex 垂直布局与极简底部栏，彻底根除遮挡问题，专注内容创作。

### 2. 强大的多页管理 (Multi-sheet)
- **多画布支持**：在一个 XMind 文件中轻松管理多个画布（Sheet）。
- **页签管理**：底部栏支持 Sheet 的快速切换、新建与删除。
- **智能缩略**：页签标题过长时自动缩略，防止布局错乱。

### 3. 多种布局自由切换
- **布局支持**：支持**思维导图 (Mind Map)**、**逻辑图 (Logic Chart)**、**组织结构图 (Org Chart)**、**树状图 (Tree Chart)** 等多种经典布局。
- **一键切换**：通过底部栏下拉菜单，实时切换当前画布的结构布局。

### 4. 智能导出 (Smart Export)
- **多格式支持**：直接将导图导出为 **PNG**、**SVG** 图片或 **Markdown** 文档。
- **智能路径定位**：
    - **工作区优先**：默认将导出文件保存到当前 **VS Code Workspace** 根目录。
    - **智能回退**：若未打开工作区，自动定位到源文件所在目录。
- **命令面板集成**：可通过 `Cmd+Shift+P` 搜索 `XMind: Export...` 快速调用导出功能。

### 5. 极致兼容性
- **非破坏性编辑**：在编辑过程中完整保留 XMind 文件的元数据，确保文件在 XMind 官方软件中打开时样式不丢失。
- **撤销/重做**：支持 `Cmd+Z` / `Cmd+Shift+Z` 快捷键，操作无忧。

## 🚀 快速上手 (Quick Start)

1. **打开文件**：双击任意 `.xmind` 文件，或将其拖入 VS Code。
2. **编辑节点**：
    - `Enter`: 插入同级节点
    - `Tab`: 插入子节点
    - `Delete`: 删除选中节点
    - `双击`: 编辑节点文本
    - `拖拽`: 自由移动节点位置
3. **管理画布**：点击底部栏 `+` 号新增 Sheet，点击页签切换。
4. **导出成果**：点击标题栏右上角按钮或底部栏布局切换右侧，一键导出。
